The Tortoise and the Hare in TLA+
An implementation of Brent's cycle detection algorithm, in PlusCal.

1.Download the TLA+ Toolbox
2.Open CycleDetection.tla file as a module
3.Set the different values for N in the TLC model checker
4.Run the TLC model checker
5.Also test invariants such as PartialCorrectness in the model checker
